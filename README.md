# Belly Button Biodiversity

An interactive data dashboard for exploring the various bacteria present on the human belly button.
This project was based on Dan Fergus's and Sarah Council's *Belly Button Biodiversity*: http://robdunnlab.com/projects/belly-button-biodiversity/

# Technologies Used
* Python (Flask)
* JavaScript (Plotly.js)
* HMTL/CSS (Bootstrap 3)
* SQLite
